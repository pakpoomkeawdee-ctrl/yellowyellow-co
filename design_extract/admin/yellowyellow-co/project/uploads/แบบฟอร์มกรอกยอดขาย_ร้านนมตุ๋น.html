🔗 ตั้งค่าการเชื่อมต่อ

1 📱 สร้าง LINE Official Account + หา Group ID (ทำครั้งเดียว)
1A — สร้างบัญชี LINE OA และ Channel Access Token
  1. เปิดเว็บ developers.line.biz → กด Log in ด้วยบัญชี LINE
  2. กด Create a new provider → ตั้งชื่อ "ร้านนมตุ๋น" → กด Create
  3. กด Create a new channel → เลือก Messaging API
  4. กรอกข้อมูล: Channel name = ร้านนมตุ๋น, Category = Food & Drink → กด Create
  5. คลิกแท็บ Messaging API → เลื่อนลงล่างสุด → Issue Channel access token → คัดลอก token ที่ได้ (เก็บไว้ใช้ใน Step 2)

1B — เพิ่ม Bot เข้ากลุ่มไลน์ + หา Group ID
  1. ในหน้า Messaging API เดิม → คัดลอก Bot basic ID (เช่น @123abcde)
  2. เปิดแอป LINE → เปิดกลุ่มไลน์ร้าน → กด ☰ → เชิญ → ค้นหา Bot ID → เพิ่มเข้ากลุ่ม
  3. กลับไปที่ developers.line.biz → ช่อง Webhook URL → วาง Make.com webhook URL ที่ได้จาก Step 2 → กด Update แล้วเปิด Use webhook: ON
  4. ใน Make.com → กด Run once (รอรับข้อมูล) → แล้วพิมพ์ข้อความอะไรก็ได้ในกลุ่มไลน์
  5. Make.com จะแสดงข้อมูลที่รับได้ → มองหา source > groupId → คัดลอกค่าที่ขึ้นต้นด้วย C (เช่น Cabc123...) → เก็บไว้ใช้ใน Step 2
💡 ถ้ากลุ่มมีสมาชิกหลายคน ต้องให้คนที่เชิญ bot เป็น Admin ของกลุ่มด้วย
2 ⚙️ ตั้งค่า Make.com (เชื่อม Google Sheets + LINE)
  1. สมัคร make.com ฟรี → กด Create new scenario
  2. เพิ่ม module แรก → ค้นหา Webhooks → เลือก Custom webhook → กด Add → Copy URL ที่ได้ไปใส่ใน Step 1B ด้วย
  3. กด ＋ → ค้นหา Google Sheets → เลือก Add a Row
    → เชื่อมบัญชี Google → เลือก Spreadsheet: ร้านนมตุ๋น → Sheet: ข้อมูลรายวัน
    → Map ข้อมูล: วันที่=date · เงินทอน=float · เงินสดรับ=cashIn · น้ำแข็ง=ice · ค่าอื่น=otherCost · รายละเอียด=costDetails · สุทธิ=net · รวมแก้ว=total · เย็น=cold · ร้อน(แก้ว)=hotG · ร้อน(ถุง)=hotB · ขวด=bottle · เหลือ=leftItems · หมด=outItems · หมายเหตุ=note · ผู้บันทึก=recorder · เวลา=timestamp
  4. กด ＋ อีกครั้ง → ค้นหา HTTP → เลือก Make a request
    URL: https://api.line.me/v2/bot/message/push
    Method: POST
    Headers: Authorization: Bearer {Channel Access Token จาก Step 1A}
                Content-Type: application/json
    Body type: Raw · Content type: application/json
    Body: 
    {"to":"GROUP_ID_จาก_Step_1B","messages":[{"type":"text","text":"{{lineMessage}}"}]}
    (แทนที่ GROUP_ID_จาก_Step_1B ด้วย groupId ที่คัดลอกมา · แทนที่ {{lineMessage}} ด้วยการเลือก lineMessage จาก dropdown ของ Make.com)
  5. กด Run once → กด บันทึก ในฟอร์ม → ตรวจสอบว่า Make.com รับข้อมูลและส่งได้ → กด OK
  6. เปิด Scheduling: ON → กด Save
💡 ดู วิดีโอสอน Make.com ถ้าติด
3 🌐 แชร์ฟอร์มให้พนักงานกรอกผ่านอินเทอร์เน็ต (Netlify)
  1. เปิดเว็บ app.netlify.com/drop
  2. ลาก-วางไฟล์ แบบฟอร์มกรอกยอดขาย_ร้านนมตุ๋น.html ลงในหน้าเว็บ
  3. รอสักครู่ → จะได้ URL เช่น amazing-shop-12345.netlify.app
  4. ส่ง URL นั้นให้พนักงานแต่ละคน → เปิดในโทรศัพท์ได้เลย
📌 ถ้าแก้ไขฟอร์ม ให้อัปโหลดไฟล์ใหม่อีกครั้ง · ข้อมูลแต่ละคนบันทึกในเครื่องตัวเอง แต่ส่งไปรวมใน Google Sheets โดยอัตโนมัติ
